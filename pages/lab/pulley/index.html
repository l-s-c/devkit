滑轮组实验

实验参数
6 N
力学数据
物体重量 G6.0 N
拉力 F6.0 N
绳子段数 n1
省力比1:1
机械效率100%
原理说明
定滑轮：固定在天花板上，只改变力的方向，不省力。F = G

滑轮与滑轮组

  • 定滑轮：轴固定不动，不省力但改变力的方向
  • 动滑轮：随物体一起移动，省一半力但不改变方向
  • 滑轮组：定滑轮+动滑轮组合，既省力又改变方向
  • n — 承担物重的绳子段数，F = G/n

滑轮组机械效率

  • η = W有用/W总 = Gh/(Fs) × 100%
  • 有用功：W有用 = G × h（提升物体）
  • 额外功：克服动滑轮重力、绳重、摩擦
  • 绳子段数 n → s = n × h

考点提醒：数绳子段数看动滑轮上有几段绳子！

典型例题

题目：用滑轮组提升 600N 重物，动滑轮重 60N，承担物重绳子 3 段，忽略摩擦，求拉力。

F = (G物 + G动) / n
F = (600 + 60) / 3
F = 220 N
