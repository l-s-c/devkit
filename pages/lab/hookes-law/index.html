胡克定律

在弹性限度内，弹簧弹力 F 与弹簧的伸长量（或压缩量）x 成正比。

  • k — 弹簧的劲度系数（N/m），由弹簧本身性质决定
  • x — 弹簧的形变量（伸长量或压缩量），x = L - L₀
  • F — 弹力（N）

注意：F = kx 中的 x 是形变量，不是弹簧长度！

弹性限度

弹簧受力超过弹性限度后，将发生永久形变，不能恢复原长。此时 F-x 关系不再是线性的。

实验中应避免超过弹性限度。当你挂了太多钩码时，弹簧变红表示已超限。

高考常考：F-x 图中超过弹性限度的部分曲线弯曲，斜率增大。

初高中衔接：弹簧测力计

初中学过的弹簧测力计，其原理正是胡克定律：弹簧伸长量与弹力成正比，所以刻度均匀。

高中的探究实验就是验证这个"成正比"的关系，并定量求出劲度系数 k。

这就是为什么弹簧测力计的刻度是均匀的！

误差分析

系统误差：弹簧自重引起的初始形变、钩码标注质量与实际质量的偏差。

偶然误差：刻度尺读数误差、弹簧未完全静止就读数。

减小误差：等弹簧静止后再读数；多次测量取平均值；用 F-x 图拟合直线代替逐次计算。

高考常考：分析 k 偏大或偏小的原因。

考点提醒：F-x 图不过原点 → 可能未扣除弹簧自重引起的初始形变，或 L₀ 测量有误。图像弯曲 → 超过弹性限度，胡克定律不再成立。

典型例题

题目：逐次挂上 50g 钩码，弹簧长度依次为 12.0, 14.0, 16.0, 18.0, 20.0 cm，原长 10.0 cm。求 k。

伸长量: x₁=2.0, x₂=4.0, x₃=6.0, x₄=8.0, x₅=10.0 cm
弹力: F₁=0.49, F₂=0.98, F₃=1.47, F₄=1.96, F₅=2.45 N
F-x 图斜率 k = ΔF/Δx = (2.45-0.49)/(0.10-0.02)
k = 1.96 / 0.08 = 24.5 N/m
