串联与并联电路

串联电路：

  • 电流处处相等：I = I₁ = I₂
  • 总电压等于各部分电压之和：U = U₁ + U₂
  • 总电阻等于各电阻之和：R = R₁ + R₂

并联电路：

  • 各支路电压相等：U = U₁ = U₂
  • 总电流等于各支路电流之和：I = I₁ + I₂
  • 总电阻：1/R = 1/R₁ + 1/R₂

串并联判断方法

  • 串联：电流只有一条路径，一个断开全部不工作
  • 并联：电流有多条路径，一个断开不影响其他
  • 电流法：从正极出发，分叉就是并联，不分叉就是串联

考点提醒：串联分压（电阻大的分到的电压大）；并联分流（电阻小的通过的电流大）。

典型例题

题目：R₁=10Ω，R₂=20Ω 串联接在 6V 电源上，求 R₁ 两端电压。

串联：R总 = R₁ + R₂ = 10 + 20 = 30Ω
I = U/R总 = 6/30 = 0.2A
U₁ = IR₁ = 0.2 × 10 = 2 V
（串联分压：U₁:U₂ = R₁:R₂ = 1:2）
