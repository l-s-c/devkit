电功率

电功率表示电流做功的快慢，即单位时间内电流所做的功。

  • P — 电功率（W）
  • P = UI = I²R = U²/R
  • 额定功率：在额定电压下工作时的功率

测小灯泡电功率

  • 用伏安法：P = UI，用电压表测 U，电流表测 I
  • 调节变阻器使灯泡在额定电压下工作
  • 高于额定电压 → 过亮 → 实际功率>额定功率 → 烧坏
  • 低于额定电压 → 偏暗 → 实际功率<额定功率

考点提醒：灯泡电阻随温度变化！温度越高电阻越大。

典型例题

题目：小灯泡额定电压 3.8V，额定电流 0.3A。求额定功率。当实际电压为 3V 时灯泡亮度如何？

P额 = U额 × I额 = 3.8 × 0.3 = 1.14 W
实际电压 3V < 额定 3.8V
灯泡偏暗，实际功率 < 额定功率
