现象观察
  • 大理石表面产生大量气泡
  • 导管口有气泡冒出
  • CO₂ 向上排出空气
  • 燃着的木条熄灭（验满）
  • 石灰水变浑浊（检验）
实验结论
1. 燃着的木条熄灭 → 证明气体是 CO₂（不支持燃烧）
2. 澄清石灰水变浑浊 → 进一步确认是 CO₂
Ca(OH)₂ + CO₂ → CaCO₃↓ + H₂O

实验原理

大理石（主要成分 CaCO₃）与稀盐酸反应，生成 CaCl₂、H₂O 和 CO₂ 气体。

为什么用大理石 + 稀盐酸？

  • 反应速度适中，便于收集
  • 不能用浓盐酸（挥发 HCl 气体，CO₂ 不纯）
  • 不能用稀硫酸（生成 CaSO₄ 微溶物覆盖大理石表面，阻止反应）
  • 不能用 Na₂CO₃（粉末反应太快，来不及收集）

为什么用向上排空气法？

  • CO₂ 能溶于水，不能用排水法
  • CO₂ 密度比空气大（相对分子质量 44 > 29），从瓶底积聚，将空气向上排出

长颈漏斗为什么要伸入液面以下？

防止生成的 CO₂ 从漏斗口逸出，保证气密性。

验满 vs 检验

  • 验满：将燃着的木条放在集气瓶口（不是伸入瓶内），木条熄灭说明已满
  • 检验：将气体通入澄清石灰水，石灰水变浑浊证明是 CO₂

O₂ vs CO₂ 制取对比（中考常考）

制取 O₂
药品：KMnO₄ 或 H₂O₂+MnO₂
收集：排水法 / 向上排空气法
验满：带火星木条 → 复燃
性质：支持燃烧，不溶于水
制取 CO₂
药品：CaCO₃ + 稀 HCl
收集：向上排空气法
验满：燃着木条 → 熄灭
检验：石灰水 → 变浑浊
性质：不支持燃烧，能溶于水
