力的平行四边形定则

两个力的合力等于以这两个力为邻边构成的平行四边形的对角线。

  • F₁、F₂ — 两个分力（由弹簧测力计测量）
  • F' — 理论合力（平行四边形对角线）
  • F — 实际合力（单力验证）
  • 若 F' ≈ F（大小和方向），则验证了定则

力是矢量，合成遵循平行四边形定则，不是简单的代数相加。

合力计算公式

合力的大小由余弦定理求得，方向由正弦定理或分量法求得。

  • θ = 0°（同向）：F = F₁ + F₂
  • θ = 180°（反向）：F = |F₁ - F₂|
  • θ = 90°（垂直）：F = √(F₁² + F₂²)

考点：合力范围 |F₁-F₂| ≤ F ≤ F₁+F₂。

实验注意事项

等效替代法：两次实验中橡皮条结点必须拉到同一位置 O 点。

角度不宜太小或太大：θ 过小时 F' 接近 F₁+F₂，误差不明显；θ 过大时橡皮条可能滑脱。

读数要准确：弹簧测力计要在拉力方向上读数，不能斜看。

高考常考：等效替代法的含义、实验误差来源分析。

误差分析

F' ≠ F 的原因：

1. 两次拉伸时结点未完全到达同一位置

2. 力的方向记录不准确（角度读数误差）

3. 弹簧测力计未校零或有系统误差

4. 作图误差（线段长度和方向的绘制精度）

考点提醒：F' 和 F 方向基本一致但大小略有差异是正常的实验结果。

典型例题

题目：F₁=3.0N，F₂=4.0N，夹角 θ=90°。求合力 F'。

F' = √(F₁² + F₂² + 2F₁F₂cosθ)
F' = √(9 + 16 + 2×3×4×cos90°)
F' = √(9 + 16 + 0) = √25
F' = 5.0 N
方向：tanα = F₂sinθ/(F₁+F₂cosθ) = 4/3
α = 53.1°（与 F₁ 的夹角）
