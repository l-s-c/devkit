电阻率公式

由电阻定律 R = ρl/S 和圆面积 S = πd²/4，可得电阻率计算公式。

  • R — 电阻丝电阻（Ω），由伏安法测量
  • d — 电阻丝直径（m），由螺旋测微器测量
  • l — 电阻丝长度（m），由刻度尺测量

注意单位换算：d 从 mm 换算到 m，l 从 cm 换算到 m。

螺旋测微器读数

读数 = 固定刻度（主尺）+ 可动刻度（副尺）× 0.01mm

主尺每格 0.5mm（上下交错），副尺 50 格转一圈 = 0.5mm，每格 0.01mm。

零误差：使用前先合拢两测量面，读取零点读数，测量值 = 读数 - 零点。

高考常考：螺旋测微器读数 + 零误差修正。

伏安法测电阻（外接法）

电阻丝阻值较小（几 Ω），适合安培表外接法（电压表并联在电阻丝两端）。

系统误差：电压表分流导致 R 测 < R 真。

高考考点：何时用内接法、何时用外接法？判据：R 大用内接，R 小用外接。

误差分析

直径测量：不同位置直径可能不一致（电阻丝不完全均匀），需多次测量取平均。

长度测量：接线柱之间的有效长度，不是电阻丝总长度。

电路测量：外接法 R 偏小 → ρ 偏小；内接法 R 偏大 → ρ 偏大。

d 的测量误差对 ρ 影响最大（ρ ∝ d²），是主要误差来源。

考点提醒：为什么 d 要测 3 次取平均？因为 ρ ∝ d²，d 的误差被平方放大。

典型例题

题目：测得 d̄=0.400mm，l=60.0cm，U-I 图斜率 R=2.39Ω。求电阻率 ρ。

d = 0.400mm = 4.00×10⁻⁴ m
l = 60.0cm = 0.600 m
S = πd²/4 = π×(4.00×10⁻⁴)²/4 = 1.257×10⁻⁷ m²
ρ = RS/l = 2.39×1.257×10⁻⁷/0.600
ρ = 5.01×10⁻⁷ Ω·m
（康铜标准值 5.0×10⁻⁷ Ω·m，吻合良好）
