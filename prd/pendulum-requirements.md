# 🔬 单摆运动模拟 — 需求文档

## 用户故事
作为学生/老师，我想通过交互式动画直观理解单摆运动规律。

## 功能需求

### Canvas 动画
- 支架 + 绳子 + 摆球绘制
- 摆球运动轨迹（淡化弧线）
- 流畅 60fps 动画

### 控制面板
- 绳长滑块（0.5m - 3m，默认 1.5m）
- 重力加速度滑块（1 - 20 m/s²，默认 9.8）
- 初始角度滑块（5° - 90°，默认 30°）
- 阻尼系数滑块（0 - 0.1，默认 0）
- 开始 / 暂停 / 重置按钮

### 拖拽交互
- 鼠标直接拖摆球到某个角度，松手释放
- 拖拽时暂停物理引擎，松手后开始运动

### 实时数据显示
- 当前角度 θ（度）
- 角速度 ω（rad/s）
- 理论周期 T = 2π√(L/g)
- 摆动次数

### 能量守恒可视化
- 动能 / 势能 / 总能量柱状图
- 实时更新，直观展示能量转换

### 公式展示
- T = 2π√(L/g) 用 KaTeX 渲染
- 简单说明小角度近似 vs 真实运动的区别

## 技术方案
- 物理模型：θ'' = -(g/L)sin(θ) - damping*ω
- 积分方法：Velocity Verlet
- 固定时间步长 dt = 1/60 + 累积器模式
- 物理计算与渲染分离
- 切 tab 回来 clamp 累积时间（max 0.1s）

## 布局
- 960px 双栏：左控制面板 300px，右 Canvas 动画区
- 移动端：单栏，Canvas 在上，控制面板在下
- 液态玻璃风格，物理蓝色系 #3B82F6
